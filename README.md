## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/globe-hoppin.git
   ```

2. Navigate to the project directory:
   ```bash
   cd globe-hoppin
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
